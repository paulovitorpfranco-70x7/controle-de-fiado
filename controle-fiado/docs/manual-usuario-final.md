# Manual do Usuario Final

## Objetivo

Este manual explica o uso diario do sistema no contexto da loja.

Ele foi pensado para operacao simples e direta.

## Perfis

### OWNER

Pode:

- consultar painel geral
- cadastrar clientes
- registrar vendas
- registrar pagamentos
- abrir cobrancas no WhatsApp

### STAFF

Pode:

- cadastrar clientes
- consultar clientes
- registrar vendas

Nao pode:

- registrar pagamentos
- acessar cobrancas
- acessar areas administrativas

## 1. Como entrar

1. Abra o link oficial do sistema.
2. Informe email e senha.
3. Toque em `Entrar`.

Se o acesso falhar:

- confira email e senha
- confirme se seu usuario esta ativo
- chame o responsavel do sistema

## 2. Navegacao

No celular:

- use a barra inferior para trocar de area
- o botao no topo serve para sair do app

No computador:

- use o menu lateral

As areas principais sao:

- `Inicio`
- `Clientes`
- `Vendas`
- `Cobrar` para `OWNER`
- `Sistema` para `OWNER`

## 3. Cadastrar cliente

1. Entre em `Clientes`.
2. Toque em `Novo cliente`.
3. Preencha nome e telefone.
4. Complete endereco ou observacoes se necessario.
5. Salve.

Boas praticas:

- sempre conferir o telefone antes de salvar
- usar o nome pelo qual a loja realmente reconhece o cliente

## 4. Consultar cliente

1. Entre em `Clientes`.
2. Use a busca por nome ou telefone.
3. Selecione o cliente desejado.

Na ficha do cliente voce consegue ver:

- saldo em aberto
- pendencias
- ultimos recebimentos

## 5. Registrar venda

1. Entre em `Vendas`.
2. Escolha o cliente.
3. Toque em `Nova venda`.
4. Preencha descricao, valor e vencimento.
5. Salve.

Antes de confirmar:

- confira se o cliente correto esta selecionado
- confira o valor final
- confira a data de vencimento

## 6. Registrar pagamento

Disponivel apenas para `OWNER`.

1. Entre em `Vendas`.
2. Escolha o cliente.
3. Toque em `Novo pagamento`.
4. Informe valor, data e forma de pagamento.
5. Se necessario, escolha qual titulo sera quitado primeiro.
6. Salve.

Depois do pagamento:

- confira se o saldo reduziu corretamente
- confira se a venda ficou `PARTIAL` ou `PAID`

## 7. Fazer cobranca pelo WhatsApp

Disponivel apenas para `OWNER`.

1. Entre em `Cobrar`.
2. Veja os clientes em `Vencem hoje` ou `Em atraso`.
3. Toque em `Enviar mensagem`.
4. Revise a mensagem.
5. Se quiser, edite o texto.
6. Toque em `Abrir no WhatsApp`.
7. Envie a mensagem no WhatsApp.

Importante:

- o envio e manual
- o sistema prepara a mensagem, mas quem envia e o operador

## 8. O que conferir todo dia

No comeco do dia:

- clientes vencendo hoje
- clientes em atraso
- saldo de clientes que fizeram pagamento recente

Durante o atendimento:

- se o cliente certo esta selecionado
- se valor e vencimento estao corretos

Ao cobrar:

- se o telefone esta correto
- se a mensagem abriu no WhatsApp certo

## 9. Erros comuns

### Cliente errado selecionado

Antes de salvar venda ou pagamento:

- confira o nome do cliente no formulario

### Telefone errado

Se o WhatsApp nao abrir corretamente:

- revise o telefone no cadastro do cliente

### Pagamento nao aparece para STAFF

Isso e esperado.

Pagamentos sao restritos ao perfil `OWNER`.

### Cobranca nao aparece para STAFF

Isso e esperado.

Cobrancas sao restritas ao perfil `OWNER`.

## 10. Quando pedir ajuda

Chame o responsavel pelo sistema se ocorrer:

- erro ao entrar
- saldo aparentemente incorreto
- venda salva no cliente errado
- pagamento nao refletido
- WhatsApp nao abre
- sistema nao carrega

## Resumo rapido

Fluxo mais comum da operadora:

1. buscar cliente
2. cadastrar cliente, se necessario
3. registrar venda

Fluxo mais comum do dono:

1. consultar cliente
2. registrar pagamento
3. cobrar no WhatsApp
